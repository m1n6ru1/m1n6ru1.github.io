Windows上自行搭建PHP套件

2022-08-02
阅读时长: 3 分钟
字数: 1293 字

安装mysql

  • 在官网下载mysql(mysql-8.0.29-winx64.zip)，解压文件夹，重命名为MySQL

  • 新建一份my.ini配置文件

    [mysqld]
# 设置3306端口
port=3306
# 设置mysql的安装目录
basedir=xxx    
# 设置mysql数据库的数据的存放目录
datadir=xxx/data   # 此处同上
# 允许最大连接数
max_connections=1000
# 服务端使用的字符集为utf8mb4
character-set-server=utf8mb4
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
[mysql]
# 设置mysql客户端默认字符集为utf8mb4
default-character-set=utf8mb4
[client]
# 设置mysql客户端连接服务端时默认使用的端口
port=3306

其中xxx是MySQL文件夹所在绝对路径

在mysql\bin目录下执行命令初始化

mysqld --initialize --console

并记住打印的临时密码

或者 mysqld --initialize-insecure --console生成空密码

安装php

  • 在PHP官网上下载线程安全版本的PHP(php-7.4.30-Win32-vc15-x64.zip)，解压文件夹，重命名为PHP

  • 在PHP文件夹内复制一份php.ini-development，重命名为php.ini，作为php的配置文件

  • 去掉extension_dir前的注释以及部分extension前的分号注释，以启用php拓展

安装apache

  • 在官网下载apache(httpd-2.4.54-win64-VS16.zip)，解压文件夹，重命名为Apache24

  • 在Apache24文件夹以外某个地方新建一个WWW文件夹用于存放网页

在Apache24/conf文件夹下的httpd.conf文件中：

  • 找到Define SRVROOT c:\apache，将其修改为Apache24文件夹所在路径

  • 找到DocumentRoot "${SRVROOT}/htdocs，将${SRVROOT}/htdocs修改为WWW文件夹所在路径

  • 找到<Directory "${SRVROOT}/htdocs">，将${SRVROOT}/htdocs修改为WWW文件夹所在路径

  • 在<IfModule dir_module>与</IfModule>之间添加如下几行：

      DirectoryIndex index.html index.php

  • 在<IfModule mime_module>与</IfModule>之间添加如下几行：

      AddType application/x-httpd-php .php
  AddType application/x-httpd-php .html
  AddType application/x-httpd-php-source .phps

  • 添加一行PHPIniDir "xxx",其中xxx是php.ini所在目录

  • 在与之间修改ScriptAlias /cgi-bin/ 后的路径为Apache24/cgi-bin/目录所在的计算机绝对路径

  • 取消LoadModule vhost_alias_module modules/mod_vhost_alias.so的注释以启用虚拟目录

  • 取消Include conf/extra/httpd-vhosts.conf前的注释

  • 添加一行 LoadModule php7_module "xxx"，xxx是php7apache2_4.dll的绝对路径

在Apache24\conf\extra文件夹下的httpd-vhosts.conf文件中添加：

<Directory "xxx">
    Options Indexes FollowSymLinks
    AllowOverride All
    Require all granted
</Directory>

<VirtualHost *:80>
    ServerAdmin localhost
    DocumentRoot "xxx"
</VirtualHost>

其中xxx是WWW文件夹所在绝对路径

配置环境变量

  • 将Apache24\bin、PHP、Mysql\bin 添加到系统环境变量中

  • 以管理员身份注册apache和mysql服务

httpd -k install 
mysqld --install
php mysql apache windows
